Currently Working On

The current web application that I am working on is called Server Side Calculator.

The requirements for the project are as follows:

  • User interface with two input fields for number values.
  • Select for mathematical operators (addition, subtraction, multiplcation, division).
  • Submit button with text "=" that will run calculations, clear button with text "C" that will clear input fields and clear history button that will remove calculations from DOM.
  • Server-side logic to run mathematical calculations.
  • Historical record to track all operations with resulting solutions.
  • Validation that does not allow logic to occur unless input fields are valid(contain numbers and are not empty fields).
